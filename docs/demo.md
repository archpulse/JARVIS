# Demo Guide

This project lands best when people can see the value immediately.

## 3-minute live demo

1. Open the app.
2. Complete first-run setup.
3. Click `INIT`.
4. Say:
   - `Check system health`
   - `Search Arch Wiki for pipewire latency`
   - `Remember that I prefer concise answers`
   - `What is on my screen?`
   - `Search for plugins related to system monitoring`

## Suggested talk track

- "This is a modular Arch Linux copilot."
- "It keeps the useful parts local and configurable."
- "It remembers user preferences and can load plugins dynamically."
- "It is designed for people who actually maintain Linux systems."

## What to show on screen

- a successful `INIT`
- a system health result
- an Arch Wiki lookup
- a memory save and recall
- a plugin search result

## Good clips to cut for social posts

- 15-second setup teaser
- 20-second Arch Wiki lookup
- 20-second system health check
- 20-second plugin search / install flow

## What makes the demo convincing

- fast start
- obvious utility
- clear output
- no hidden magic
- extensible plugin architecture
